Project Title

Smart Industrial Sensor Hub using MQTT, Python and SQLite

Description

Developed a Smart Industrial Sensor Hub for real-time monitoring of temperature and humidity using Python and MQTT communication. Implemented a live dashboard using Tkinter, sensor data logging using CSV and SQLite databases, real-time alarm monitoring, analytics visualization with Matplotlib, and automated PDF report generation. The project simulates Industrial IoT monitoring systems used in manufacturing and automation environments.
