# Smart Industrial Sensor Hub

A professional Industrial IoT monitoring system designed to simulate real-world factory automation and predictive maintenance workflows.

This project demonstrates the integration of embedded systems, IoT communication, cloud technologies, machine learning, and software testing within a single end-to-end architecture.

## Key Features

* STM32-based embedded firmware architecture
* FreeRTOS multitasking and task scheduling
* MQTT-based real-time communication
* Python gateway and data processing
* TinyML anomaly detection for predictive maintenance
* Cloud integration for remote monitoring
* Automated testing and validation
* Real-time dashboard visualization
* Modular and scalable industrial design

## Technology Stack

* STM32
* Embedded C
* FreeRTOS
* MQTT
* Python
* Flask
* NumPy
* Pandas
* Matplotlib
* TinyML
* GitHub Actions
